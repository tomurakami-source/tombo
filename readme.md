## 3. 5セクション構成（Golden 5-Section Architecture）

### Section 1: Hero

- **ビジュアル:** 15秒ライブ切り抜き動画を全画面ループ再生（`muted`, `autoPlay`, `loop`）
- **キャッチコピー:** 「平成の熱狂を、令和のビートで。新しいロックンロールの夜明け。」
- **狙い:** 聴覚（音）と視覚（レトロな質感）で一瞬で心を掴む

### Section 2: Live Info

- **ビジュアル:** 直近ライブ1件をカード形式で強調表示
- **UI:** 画面内で最も目立つ「予約する」ボタン（スクロール追従）
- **導入文:**
  > 「ライブハウスは、扉を開けるまでが一番勇気がいる。でも、その先にはあなたを肯定する爆音が待っている。ザ・トンボバンチは、初めての人も、一人の人も、女性も、等しく熱狂できる場所を約束します。」

### Section 3: Audio/Video

- **ビジュアル:** YouTube埋め込み（高速化のためLiteプレイヤー使用）
- **導入文:**
  > 「泥臭いソウルミュージックのグルーヴと、鋭利な日本のロックンロールが衝突する。懐かしいのに、誰も聴いたことがない。そんな『新しいスタイルのロックンロール』を、まずはその耳で確かめてほしい。」

### Section 4: Ticket Form

- **仕様:** 外部サイト遷移なし、LP内完結フォーム
- **項目:** 名前、メールアドレス、予約枚数
- **バックエンド:** 送信完了をトリガーにGAS経由でボーカルのLINEへ通知
- **導入文:**
  > 「予約は、名前と枚数を入れるだけ。わずか10秒で完了。チケット代は当日、会場の受付で。スマートに、確実に。僕たちの夜を、今すぐ予約して。」

### Section 5: Concept/Bio

- **内容:** ボーカルのキャリア（CRACK BANQUET、C.メアリー&15等）へのリスペクトと新バンドの志
- **本文:**
  > 「2022年活動開始。日本のロックシーンを駆け抜けたVo/Gが、再び東京の夜に火を灯す。ルーツにあるのは、黄金時代の日本のロックと、深いソウルミュージック。過去の模倣ではない。今を生きる僕たちのための『新しいロックンロール』を探求し、ザ・トンボバンチは鳴り続ける。」

## 4. Q&A（アクセシビリティ・不安解消）

フッター付近に配置。

**Q. ライブハウスって怖くないですか？**
A. 暗くて地下にあるので、そう感じるかもしれません。でもザ・トンボバンチのライブは、デジタルアートとロックが融合した洗練された空間です。

**Q. 一人で行っても大丈夫ですか？**
A. もちろんです。実は、来場者の約半分が「一人参戦」。音楽に没頭したいあなたを、僕たちは歓迎します。

**Q. 何を持っていけばいい？**
A. スマホの予約確認画面と、少しの好奇心、それだけで十分です。

## 5. 技術スタック

| 項目 | 内容 |
|------|------|
| Framework | Next.js (App Router) |
| Styling | Tailwind CSS |
| Animation | Framer Motion（スクロール演出） |
| 画像最適化 | `next/image`、WebP/WebM形式 |
| バックエンド連携 | Google Apps Script (GAS) へのPOST通信（予約管理・LINE通知） |
| トラッキング | Google Analytics 4 (GA4)、`?ref=` パラメータによる流入元分析 |
| 目標スコア | Lighthouse Performance Score 95+ |

## 6. KPI（成功の指標）

| 指標 | 定義 |
|------|------|
| 予約完了数 | LP経由の予約数（ライブハウスへの直接連絡は除外） |
| CVR | LP閲覧者のうち予約に至った割合 |
| SNS Effective Rate | X/Instagram「フォローバック活動」から予約に繋がった人数 |
