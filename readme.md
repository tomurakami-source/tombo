🎸 The Tonbo Bunch Official LP Project
"Neo-Heisei Soul Rock" - 集客をハックするエンジニア・プロデュース
ザ・トンボバンチの「ロック×ソウル×平成レトロ」という唯一無二の世界観を、現代のWeb技術で最速かつスマートに届けるためのランディングページ。

🎯 プロジェクトの目的
コンバージョン（予約）の最大化:
「ライブハウスは怖い・行きにくい」という心理的障壁を、洗練されたUXと情報の透明性で排除する。

ターゲット層の拡大:
ボーカルの旧来ファンを大切にしつつ、20代・女性・一人客が「ここなら安心、かつカッコいい」と思えるデジタルアート空間を提示する。

データ駆動のプロデュース:
エンジニアリング（計測）に基づき、SNS活動から予約までの導線を可視化・改善する。

🛠 黄金の5セクション構成 (Golden 5-Section Architecture)
ユーザーの熱量を逃さず、かつ「納得」させて予約へ導くための絶対的な階層構造です。

【Hero】 視覚的インパクトの瞬発力

内容: 「最強の15秒動画」を全画面ループ再生。

演出: キャッチコピー「平成レトロ×新しいロックンロール」をオーバーレイ。

狙い: 聴覚（音）と視覚（レトロな質感）で一瞬で心を掴む。

【Live Info】 迷わせない導線

内容: 直近のライブ1件をカード形式で強調。

演出: 「予約する」ボタンを画面内で最も大きく、押しやすく配置。

狙い: 興味を持った瞬間に「次、いつどこに行けばいいか」を提示する。

【Audio/Video】 音楽的信頼の構築

内容: YouTubeのメインMVを埋め込み。

狙い: 「ロックとソウルの融合」をフル尺で納得させ、予約への確信を与える。

【Ticket Form】 離脱率ゼロの予約体験

内容: LP内に直接埋め込まれた独自の予約フォーム。

実装: 外部サイトに飛ばさず、名前と枚数だけで完了する最小UX。

エンジニア・ハック: 送信完了をトリガーにGAS経由でボーカルのLINEへ通知。

【Concept/Bio】 魂の継承と更新

内容: ボーカルのキャリア（CRACK BANQUET等）へのリスペクト。

狙い: 過去の資産を「古いもの」ではなく「エモい資産」として再定義し、信頼を築く。

🎨 デザイン・コンセプト
GLIM SPANKYをベンチマークとした「Neo-Heisei Soul Rock」

Texture: フィルムグレイン（粒子感）、微細なノイズ、VHSのようなグリッチ演出。

Color: 煤けたオレンジ、深いネイビー、セピアがかったホワイト。

Typography: 平成初期の雑誌のような大胆なセリフ体と、モダンなゴシック体の対比。

UX: 「地下の暗闇」を「洗練されたデジタルアート空間」へ変換。

💻 技術スタック & 実装仕様
Framework: Next.js (App Router)

Styling: Tailwind CSS + Framer Motion（滑らかなスクロール演出）

Performance: Lighthouse Performance Score 95+ を目標。モバイルでの爆速表示を徹底。

Tracking: Google Analytics 4 (GA4)。?ref=x などのパラメータでSNS別流入を計測。

Accessibility:

ライブハウスまでの「道のりチュートリアル」の掲載。

スマホ下部に常に追従する「Sticky Reservation Button」。

📈 成功の指標 (KPI)
予約完了数: ライブハウスへの直接連絡ではなく、LP経由の予約を主要指標とする。

CVR (Conversion Rate): LP閲覧者のうち、どれだけが予約に至ったか。

SNS Effective Rate: XやInstagramの「フォローバック活動」が、実際に何人の予約を生んだかの可視化。
